BioPacific Laboratory Information System

2023-Present

A Laboratory Information Management System (LIMS) is specialized software developed to efficiently handle the substantial amounts of data generated during experimental processes. At its core, the LIMS functions as a tracking tool that oversees the journey of a sample throughout its different phases of production and analysis.

Project Details

As a Scientific Programmer and Designer, I developed object-oriented, multi-tier web applications using Django, focusing on user-friendly interface design and resolving frontend issues. By incorporating user feedback, I boosted adoption by 50% and enhanced usability. I also created data analysis and visualization tools, simplifying complex datasets with statistical algorithms. In collaboration with a Database Developer, I proposed APIs for backend processing and engineered a new desktop app, leading testing and debugging efforts for multiple projects using MySQL.

Techniques & Tools

  • CSS, HTML, Python web design
  • Javascript testing and maintanence
  • SQL data modeling and querying
  • Svelte kit frontend maintance

Challenges

Working with SQL can be a powerful tool, but it's also a bit of a nightmare at times. While developing the BPM LIMS system, one of the biggest challenges was taming SQL queries for efficient data retrieval and managing the sheer complexity of relational databases. There were moments when it felt like the simplest query had a mind of its own, returning unexpected results or causing performance bottlenecks.
